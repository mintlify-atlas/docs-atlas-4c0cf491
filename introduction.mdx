## What is Neru?

## Works Everywhere

Neru works seamlessly across all major macOS applications:

| Category | Apps |
| --- | --- |
| **Native macOS** | Finder, Safari, Mail, System Settings, Terminal |
| **Electron** | VS Code, Cursor, Slack, Spotify, Obsidian, Discord |
| **Browsers** | Chrome, Firefox, Safari, Arc, Brave, Zen |
| **Creative** | Figma, Adobe Illustrator, Photoshop |
| **System UI** | Menubar, Dock, Mission Control, Notification Center |

<Note>
Grid mode works universally. Hints mode works where the accessibility tree is exposed. See the [App Compatibility](/advanced/app-compatibility) guide for details.
</Note>

## Why Neru?

<AccordionGroup>
  <Accordion title="Free & Open Source">
    Unlike paid alternatives (Homerow, Mouseless, Wooshy), Neru is completely free with no paywalls or subscriptions. MIT licensed.
  </Accordion>
  
  <Accordion title="Endlessly Customizable">
    Configure everything through a single TOML file: keybindings, colors, behaviors. Version-control your config in dotfiles.
  </Accordion>
  
  <Accordion title="Cross-Platform Ready">
    Built with hexagonal architecture from day one. macOS is stable, Linux & Windows support in progress.
  </Accordion>
  
  <Accordion title="Active Development">
    Unlike discontinued alternatives (Shortcat) or unmaintained projects (Vimac), Neru is actively maintained.
  </Accordion>
</AccordionGroup>

## Quick Start

Get up and running in under 2 minutes:

<Steps>
  <Step title="Install via Homebrew">
    ```bash
    brew tap y3owk1n/tap
    brew install --cask y3owk1n/tap/neru
    ```
  </Step>
  
  <Step title="Grant Accessibility Permission">
    Open **System Settings â†’ Privacy & Security â†’ Accessibility** and enable **Neru**
  </Step>
  
  <Step title="Launch Neru">
    ```bash
    open -a Neru
    ```
  </Step>
  
  <Step title="Try Navigation">
    Press `Cmd+Shift+C` for Recursive Grid mode, or `Cmd+Shift+Space` for Hints mode
  </Step>
</Steps>
